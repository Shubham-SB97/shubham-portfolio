Project Name : Varad Automation
CCTV Surveillance & Security System

Description: The CCTV Surveillance & Security System is a comprehensive web-based solution that provides real-time monitoring, recording, and management of CCTV footage for residential, commercial, and industrial sectors. Built using Spring Boot and Angular 18, it ensures scalability, security, and high performance. The system supports role-based access control, motion detection alerts, and integrates additional services like CCTV sales, EPBX systems, biometric attendance devices, and solar installations under government schemes.

Technology Stack:
✅ Backend: Java, Spring Boot, Spring Security, REST API
✅ Frontend: Angular 18, TypeScript, Bootstrap, HTML, CSS
✅ Database: MySQL (stores user & employee data, camera metadata, and video records )
✅ API Testing Tool: Postman ✅ Duration : 30 - 50 Days.
